Topik: Logika Percabangan (If / Else If / Else)

Program ini meminta pengguna memasukkan kode shift kerja dalam bentuk angka:

  • 1 = Pagi
  • 2 = Siang
  • 3 = Malam

Sistem akan membaca input menggunakan prompt(), kemudian mengevaluasi menggunakan logika if / else if / else. Hasilnya akan ditampilkan pada Console Browser.

Untuk melihat hasilnya, tekan F12 atau Fn + F12, lalu pilih tab Console.
