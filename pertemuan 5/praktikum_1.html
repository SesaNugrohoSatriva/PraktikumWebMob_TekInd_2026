Membuat Function Perhitungan Daya

Program ini menggunakan JavaScript untuk menghitung daya listrik dengan rumus P = V × I, di mana:

  • P = Daya (Watt)
  • V = Tegangan (Volt)
  • I = Arus (Ampere)
Penjelasan Kode JavaScript

Fungsi hitungDaya(tegangan, arus) digunakan untuk menghitung daya berdasarkan nilai tegangan dan arus. 

function hitungDaya(tegangan, arus){
    let daya = tegangan * arus;
    return daya;
}

Nilai yang digunakan dalam program:

  • Tegangan Mesin = 220 Volt
  • Arus Mesin = 10 Ampere
Hasil perhitungan dapat dilihat melalui Console Browser (F12 → Console).
