Step 1 – Variabel dan Tipe Data


let namaMesin = "CNC-Mazak-01";
let targetHarian = 500;

console.log("Mesin: " + namaMesin);
console.log("Target Harian: " + targetHarian);

Variabel namaMesin bertipe String karena menyimpan teks. Variabel targetHarian bertipe Number karena digunakan untuk perhitungan matematis. Nilai ditampilkan menggunakan console.log() ke console browser.

Step 2 – Operator Aritmatika


let produksiPagi = 200;
let produksiSiang = 150;

let totalProduksi = produksiPagi + produksiSiang;
let kekurangan = targetHarian - totalProduksi;

console.log("Total Produksi: " + totalProduksi);
console.log("Kekurangan Target: " + kekurangan);

Total produksi dihitung dengan operator penjumlahan (+). Kekurangan target dihitung menggunakan operator pengurangan (-). Hasil menunjukkan selisih antara target dan pencapaian aktual.

Step 3 – Logika If / Else (Status Maintenance)


let jamOperasional = 1250;
let batasMaksimal = 1200;

if (jamOperasional >= batasMaksimal) {
    console.log("Status: MAINTENANCE WAJIB");
} else if (jamOperasional > 1000) {
    console.log("Status: SIAP HATI-HATI");
} else {
    console.log("Status: BERJALAN NORMAL");
}

Struktur percabangan digunakan untuk mengevaluasi kondisi. Jika jam operasional melebihi batas maksimal, maka mesin wajib maintenance. Jika mendekati batas, sistem memberi peringatan.

Step 4 – Perhitungan Availability


let jamKerjaPlanned = 8;
let jamKerjaAktual = 6.5;

let availability = (jamKerjaAktual / jamKerjaPlanned) * 100;
availability = availability.toFixed(2);

console.log("Availability: " + availability + "%");

if (availability >= 90) {
    console.log("Kategori: WORLD CLASS");
} else if (availability >= 80) {
    console.log("Kategori: BAIK");
} else {
    console.log("Kategori: BURUK");
}

Availability dihitung menggunakan rumus rasio waktu aktual terhadap waktu rencana dikali 100%. Fungsi toFixed(2) membatasi dua angka di belakang koma. Nilai kemudian dikategorikan menggunakan percabangan.

Step 5 – Input Pengguna (prompt dan alert)


let namaOperator = prompt("Masukkan Nama Operator:");
let shiftKerja = prompt("Masukkan Shift (Pagi/Siang/Malam):");

if (shiftKerja === "Malam") {
    alert("Halo " + namaOperator + 
          ", Shift malam mendapat tambahan Rp 20.000.");
} else {
    alert("Halo " + namaOperator + 
          ", Selamat bekerja.");
}

Fungsi prompt() digunakan untuk menerima input dari pengguna. Nilai yang dimasukkan akan disimpan dalam variabel namaOperator dan shiftKerja.

Nilai yang dikembalikan oleh prompt() selalu bertipe String. Jika pengguna menekan Cancel, maka nilainya adalah null. Jika kosong, maka nilainya adalah String kosong ("").

Struktur if membandingkan isi shiftKerja dengan teks "Malam". Jika sama persis (menggunakan operator ===), maka sistem memberikan notifikasi tambahan uang makan.
