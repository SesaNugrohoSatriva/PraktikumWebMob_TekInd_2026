Simulasi Antrian Pekerjaan Mesin

Program ini mensimulasikan sistem antrian pada lingkungan job shop menggunakan array of object. Setiap job memiliki atribut idJob, namaProses, dan durasi (menit). 

let antrianMesin = [
    { idJob: "J01", namaProses: "Drilling", durasi: 30 }
];

Function prosesAntrian() digunakan untuk menampilkan setiap job dalam antrian menggunakan perulangan for. 

function prosesAntrian(antrianMesin) {
    for (let i = 0; i < antrianMesin.length; i++) {
        let job = antrianMesin[i];
        console.log(
            "Memproses Job " + job.idJob +
            " - " + job.namaProses +
            " selama " + job.durasi + " menit"
        );
    }
}

Setelah itu, program menambahkan job baru ke dalam array menggunakan method push(), kemudian function dipanggil kembali untuk menampilkan daftar terbaru.
