Penggunaan Object dalam JavaScript

Program ini menggunakan struktur data Object untuk merepresentasikan data sebuah mesin produksi. Object memungkinkan penyimpanan data dalam bentuk pasangan key : value. 

let mesinA = {
    id: "M-01",
    nama : "Injection Molding 50 Ton",
    merk : "Toshiba",
    tahunBeli : 2018,
    status : "Running",
    spesifikasi : {
        kecepatanMax : 200,
        tekananMax : 150
    }
};

Data di dalam object dapat diakses menggunakan tanda titik (dot notation), misalnya: 

mesinA.nama
mesinA.status
mesinA.spesifikasi.tekananMax

Program juga mensimulasikan perubahan kondisi mesin dengan mengubah nilai properti status dari Running menjadi Breakdown.
