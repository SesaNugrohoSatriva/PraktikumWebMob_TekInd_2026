Praktikum JavaScript Dasar

Petunjuk

Buka Developer Tools dengan menekan F12 atau Fn + F12 , lalu pilih tab Console. Semua hasil perhitungan dan logika program akan tampil di sana.

Materi yang Dipraktikkan

  • Variabel dan Tipe Data (String, Number)
  • Operator Aritmatika (Penjumlahan dan Pengurangan)
  • Logika If / Else If / Else
  • Perhitungan Availability (%)
  • Validasi Status Maintenance Mesin
  • Input Pengguna dengan prompt()
  • Output dengan console.log() dan alert()

Penjelasan Program

Program ini mensimulasikan sistem monitoring mesin produksi. Sistem menghitung total produksi, mengevaluasi kekurangan target, mengecek status maintenance berdasarkan jam operasional, serta menghitung availability produksi.

Di bagian akhir, program meminta input nama operator dan shift kerja. Jika operator bekerja pada shift malam, maka akan muncul notifikasi tambahan uang makan.
