Penggunaan Array dan Looping dalam JavaScript

Program ini menggunakan struktur data Array untuk menyimpan daftar kode cacat produksi. Data awal yang digunakan: 

let daftarCacat = ["C-001", "C-005", "C-012", "C-001", "C-020"];

Panjang array dapat diketahui menggunakan properti length, kemudian ditampilkan ke console.

Selanjutnya dilakukan perulangan menggunakan for untuk menampilkan setiap item dalam array. 

for (let i = 0; i < daftarCacat.length; i++){
    console.log("Item ke-", (i+1) + " : ", daftarCacat[i]);
}

Program juga menambahkan data baru menggunakan method push(), sehingga jumlah data bertambah.
