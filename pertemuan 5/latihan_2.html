Menghitung Jumlah Kemunculan Data dalam Array

Program ini bertujuan untuk menghitung berapa kali kode cacat "C-001" muncul dalam sebuah array menggunakan perulangan for dan struktur kondisi if. 

let daftarCacat = ["C-001", "C-005", "C-012", "C-001", "C-020"];

let jumlahC001 = 0;

for (let i = 0; i < daftarCacat.length; i++) {
    if (daftarCacat[i] === "C-001") {
        jumlahC001++;
    }
}

Variabel jumlahC001 digunakan sebagai counter. Setiap kali elemen array bernilai "C-001", maka nilai counter akan bertambah satu.
