Program ini meminta pengguna memasukkan kode shift kerja dalam bentuk angka:

  • 1 = Pagi
  • 2 = Siang
  • 3 = Malam

Sistem membaca input menggunakan prompt(), kemudian mengevaluasi menggunakan logika if / else if / else. Hasil ditampilkan melalui console.log().

Step 1 – Mengambil Input dari Pengguna


let input = Number(
    prompt("Masukkan kode shift (1=Pagi, 2=Siang, 3=Malam)")
);

Fungsi prompt() digunakan untuk meminta input dari pengguna. Nilai yang dikembalikan oleh prompt() bertipe String.

Fungsi Number() digunakan untuk mengonversi nilai String menjadi tipe Number agar dapat dibandingkan menggunakan operator perbandingan ketat (===).

Jika pengguna menekan Cancel, maka nilai menjadi null. Jika input bukan angka, maka hasil konversi menjadi NaN.

Step 2 – Logika Percabangan If / Else If / Else


if (input === 1) {
    console.log("Shift Anda adalah Pagi");
} else if (input === 2) {
    console.log("Shift Anda adalah Siang");
} else if (input === 3) {
    console.log("Shift Anda adalah Malam");
} else {
    console.log("Shift Tidak Valid");
}

Struktur if / else if / else digunakan untuk mengevaluasi kondisi secara berurutan.

Operator === disebut strict comparison. Operator ini membandingkan nilai dan tipe data sekaligus. Karena input sudah dikonversi menjadi Number, maka perbandingan akan berjalan dengan akurat.

Jika nilai tidak termasuk 1, 2, atau 3, maka sistem menampilkan pesan "Shift Tidak Valid".

Alur Logika Program

  1. Program meminta pengguna memasukkan kode shift.
  2. Input dikonversi menjadi Number.
  3. Sistem membandingkan nilai input dengan 1, 2, dan 3.
  4. Jika cocok, nama shift ditampilkan.
  5. Jika tidak cocok, sistem menampilkan pesan tidak valid.
Jalankan program dan lihat hasilnya melalui Console Browser (F12).
