Function Perhitungan Lingkaran

Program ini menggunakan fungsi hitungLingkaran() untuk menghitung luas dan keliling lingkaran berdasarkan nilai jari-jari.

Rumus yang digunakan:

  • Luas = π × r²
  • Keliling = 2 × π × r
function hitungLingkaran(jariJari) {
    const phi = Math.PI;
    const luas = phi * jariJari * jariJari;
    const keliling = 2 * phi * jariJari;

    return {
        luas: luas,
        keliling: keliling
    };
}

Fungsi mengembalikan object yang berisi nilai luas dan keliling. Nilai tersebut kemudian ditampilkan pada console dengan pembulatan dua angka desimal menggunakan toFixed(2).

Hasil perhitungan dapat dilihat melalui Console Browser (F12 → Console).
