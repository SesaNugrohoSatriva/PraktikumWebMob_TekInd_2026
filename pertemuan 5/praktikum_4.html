Array of Object dan Logika Re-Order

Program ini menggunakan array yang berisi object untuk merepresentasikan data material di gudang. Setiap object memiliki properti kode, nama, stock, dan satuan. 

let gudangMaterial = [
    { kode: "MAT-01", nama: "Baja Ringan", stock: "50", satuan:"Batang"},
    { kode: "MAT-02", nama: "Plastik ABS", stock: "120", satuan:"Kg"},
    { kode: "MAT-03", nama: "Oli Mesin", stock: "10", satuan:"Liter"}
];

Iterasi dilakukan menggunakan method forEach() untuk menampilkan setiap data material. 

gudangMaterial.forEach(function(item){
    console.log(item.kode + " - " + item.nama + 
                " : " + item.stock + " " + item.satuan);
});

Di dalam perulangan terdapat logika Re-Order. Jika stok kurang dari 20, maka sistem memberikan peringatan untuk segera melakukan Purchase Order (PO).
